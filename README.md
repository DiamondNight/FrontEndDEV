# Selectores BASICOS
* De Tipo      -   div {...}
* De Clase     -   . elemento {...}
* De Id        -   #id-del-elemento {...}
* De Atributo  -   a [href=".."] {...}
* Universal    -   * {...}

* Descendientes | Div p
* Hijo Directo | Div > p
* Elemento Adyacente | Div + p
* General de Hermanos | Div ~ p
